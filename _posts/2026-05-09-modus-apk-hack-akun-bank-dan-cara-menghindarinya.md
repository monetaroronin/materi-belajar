## Apa Itu Social Engineering?

**Social engineering** adalah taktik manipulasi psikologis yang digunakan penjahat agar korban mau memberikan informasi sensitif atau melakukan sesuatu tanpa disadari.

## Modus Penipuan APK

Penjahat mengirim file APK melalui WhatsApp, Telegram, atau SMS dengan berbagai kedok:

- **Kurir paket** — "Kak, ini cek resi paketnya." → disertai file APK
- **Undangan pernikahan** — "Mohon hadir ya." → file APK berkedok undangan
- **Hadiah/undian** — "Selamat Anda menang hadiah, klik di sini untuk klaim." → download APK
- **Info pemerintah/bank** — "Tagihan BPJS belum dibayar" atau "Rekening akan diblokir, update data perbankan." → file APK
- **Surat tilang** — "Anda kena tilang, cek detail di sini." → link download APK

> **Pola umum:** Penjahat membuat korban panik atau penasaran sehingga tanpa pikir panjang mengklik dan menginstal file tersebut.

## Cara Kerja APK Penyadap

1. Korban menginstal APK yang dikirim penjahat.
2. Aplikasi meminta izin akses **baca SMS**.
3. Korban mengklik "Izinkan" tanpa membaca.
4. Aplikasi berjalan **diam-diam di background** — tidak ada ikon, tidak ada notifikasi.
5. Setiap SMS masuk otomatis **dikirim ke server penjahat**.
6. Penjahat menggunakan nomor HP korban untuk login ke **mobile banking / internet banking** dari HP lain.
7. Bank mengirim **kode OTP lewat SMS** ke HP korban.
8. SMS OTP tersadap → dikirim ke penjahat.
9. Penjahat memasukkan OTP → **berhasil masuk ke rekening korban**.
10. Penjahat mentransfer atau menguras seluruh saldo — **dalam hitungan menit**.

## Cara Menghindari Penipuan APK

### 1. Jangan Pernah Instal APK dari Chat
Tidak ada kurir yang menyuruh instal aplikasi untuk cek resi. Tidak ada undangan pernikahan berbentuk APK. Jika ada yang mengirim file atau meminta instal aplikasi lewat chat → **100% penipuan**.

### 2. Perhatikan Ekstensi File
Jika file yang dikirim berekstensi **.apk**, jangan diklik. Langsung hapus atau blokir nomornya, terutama jika pengirim tidak dikenal.

### 3. Jangan Beri Izin Sembarangan
Periksa apakah izin yang diminta masuk akal. Contoh:
- Aplikasi cek resi minta akses SMS → **tidak masuk akal**
- Aplikasi undangan minta akses SMS → **tidak masuk akal**

Jika tidak masuk akal, jangan instal atau langsung hapus aplikasinya.

### 4. Gunakan Nomor HP Berbeda untuk Akun Bank
Miliki nomor khusus yang **hanya dipakai untuk akun bank**, tidak digunakan untuk chat sehari-hari, dan tidak dibagikan ke siapapun. Dengan begitu, jika HP utama yang dipakai chat terkena hack, nomor yang terdaftar di bank tetap aman.

### 5. Aktifkan Semua Fitur Keamanan di Aplikasi Bank
Beberapa fitur yang perlu diaktifkan:
- **Limit transaksi**
- **Notifikasi real-time**
- **Verifikasi biometrik** (fingerprint atau Face ID)
